隱私權政策 (Privacy Policy)

最後更新日期：2026年5月

歡迎您使用「∑ Calc - 超強萬能計算機」（以下簡稱「本網站」）。我們非常重視您的隱私權。本隱私權政策旨在說明我們如何處理、收集與保護您在使用本網站時所產生的資訊。

1. 資料收集與儲存 (Data Collection and Storage)

本網站的核心功能（包含計算機、單位換算與公式計算）完全在您的瀏覽器端（客戶端）執行。

  • 本地儲存 (Local Storage)：為了提供更好的使用者體驗，本網站會使用您瀏覽器的 localStorage 功能來保存您的偏好設定（如：深色/淺色主題、多國語言選擇）、計算歷史紀錄以及「自建公式」。這些資料僅儲存於您的個人設備中，本網站並無設置後端資料庫，也不會主動收集、上傳或將這些個人資料傳送至任何外部伺服器。
  • 無需註冊：使用本網站的任何功能均無需註冊帳號或提供任何個人身分識別資訊 (PII)。

2. 第三方服務與 API (Third-party Services and APIs)

為了提供完整的服務，本網站有串接部分第三方服務，這些服務可能會收集部分技術資訊：

  • ExchangeRate-API：本網站的「即時匯率換算」功能依賴於 ExchangeRate-API 取得最新匯率。當您使用此功能或點擊更新匯率時，您的裝置將會向該 API 發送網路請求，該服務供應商可能會依法規要求記錄您的 IP 位址等基礎連線資訊。
  • Google AdSense (廣告投放)：本網站未來可能會使用 Google AdSense 來展示廣告。
    • 第三方供應商（包括 Google）會使用 Cookie 來放送廣告。這些廣告是根據使用者先前對本網站或網際網路上其他網站的造訪紀錄來投放。
    • Google 及其合作夥伴使用廣告 Cookie，即可根據使用者對本網站和/或網際網路上其他網站的造訪紀錄，向使用者放送合適的廣告。
    • 使用者可前往 廣告設定 選擇停用個人化廣告。您也可以前往 www.aboutads.info 選擇停用第三方供應商用於個人化廣告的 Cookie。

3. 分析工具 (Analytics)

本網站可能使用 Google Analytics 等流量分析工具來了解網站的整體使用狀況，以改善我們的服務。這些工具會使用 Cookie 收集匿名的網站流量數據，包含瀏覽頁面、停留時間與裝置類型等，但不會用來追蹤或識別您的真實身分。

4. 外部連結 (External Links)

本網站可能包含其他網站的連結。如果您點擊這些連結離開本網站，請注意，我們對其他網站的隱私權實務不負任何責任。建議您在提供任何個人資訊前，先閱讀該網站的隱私權政策。

我們保留隨時修改本隱私權政策的權利。如有任何重大變更，我們將在網站首頁或此頁面上更新修改日期。建議您定期檢閱本政策，以了解我們如何保護您的資訊。
