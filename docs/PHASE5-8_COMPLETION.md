## 📐 Phase 5：微積分可視化

### 三項工具

1. **切線顯示工具 (Tangent)**
   - 點擊曲線顯示該點切線
   - 自動計算數值微分 f'(x)（中央差分法，h=1e-5）
   - 切點以實心圓標示
   - 標籤格式：`f'(x.xx) = slope.xxx`
   - 同點再次點擊清除切線

2. **積分區域著色 (Integral)**
   - 點擊兩點選擇積分區間 [a, b]
   - 自動計算 Simpson 1/3 法則積分（n=500）
   - 著色曲線下方區域（半透明）
   - 顯示積分值 `∫=xxx.xxx`
   - 邊界垂直線標示積分範圍

3. **斜率場 (Slope Field)**
   - 視覺化函數的微分場
   - 22×14 格點顯示斜率方向短線
   - 對任一可見函數啟用
   - 適合理解微分方程動力學

### 數值方法
```javascript
numericalDerivative(fn, x, params, h=1e-5)  // 中央差分
numericalIntegral(fn, a, b, params, n=1000) // Simpson 1/3
newtonRaphson(fn, x0, params)               // 求根
```

### 驗證結果
- ✅ x² 在 x=2 的切線斜率 = **4.000**（理論 2x|x=2 = 4 ✓）
- ✅ ∫₋₂³ x² dx = **11.667**（理論 35/3 ≈ 11.6667 ✓）
- ✅ 斜率場在 x=0 附近水平，向兩側遞增（符合 2x 趨勢 ✓）

## ⊕ Phase 6：方程求解器

### 功能：找多函數交點

點擊「⊕ 交點」按鈕後：
1. 在當前 X 範圍內以 800 點掃描所有可見函數對
2. 偵測 sign change（包括精確零點）
3. 用 **Bisection 二分法** 精化到 1e-9 精度
4. 自動去重（避免相鄰點重複）
5. 標記所有交點為橘色圓點

### 演算法亮點
```javascript
refinePairBisection(fA, fB, lo, hi, params)
// 對 g(x) = fA(x) - fB(x) 做二分法
// 收斂條件：|g(mid)| < 1e-9 或 (hi-lo)/2 < tol
```

### 驗證結果
測試案例：x² = x + 6
- ✅ 找到精確交點 **(-2, 4)** 和 **(3, 9)**
- ✅ 解析解：x²-x-6=0 → x=(1±5)/2 → x∈{-2, 3} 完美吻合

## 📊 Phase 7：統計圖表

### 三種統計可視化

#### 1. 直方圖 (Histogram)
- 自動分箱（預設 12 bins）
- 顯示每箱頻數
- 統計摘要：n / mean / σ / min / max
- 適合單變量分布視覺化

#### 2. 散佈圖 + 線性迴歸 (Scatter + Regression)
- 最小平方法線性迴歸
- 自動繪製迴歸線
- 顯示方程式：`y = mx + b`
- 計算決定係數 **R²**
- 適合相關性分析

#### 3. 盒狀圖 (Box Plot)
- 計算五數摘要：min, Q1, median, Q3, max
- 顯示鬚線（whiskers）至 1.5×IQR 範圍
- 自動標示離群點（紅色點）
- 統計：n, IQR, 離群點數量

### 資料輸入格式
```
單列數值（直方圖/盒狀圖）：
1.5, 2.3, 3.1, 4.2, ...

x,y 對組（散佈圖）：
0,1; 1,3; 2,5; 3,7; ...
```

### 驗證結果
- ✅ 直方圖：N(5, 2) 樣本 200 個 → mean=5.052, σ=1.955（精準）
- ✅ 散佈圖：y=2x+3 + 噪音 → 迴歸 y=1.999x+3.136, **R²=0.9962**
- ✅ 盒狀圖：50 正常 + 2 離群點 → 正確識別 outliers=2

## 🎲 Phase 8：3D 表面繪圖

### 自製輕量 3D 引擎（無外部依賴）

不使用 WebGL/Three.js，而是用純 Canvas 2D 加自製 3D→2D 投影：

```javascript
project3D(x, y, z, state, w, h)
// 1. X 軸旋轉（仰角 rotX）
// 2. Y 軸旋轉（方位 rotY）
// 3. 正交投影（不含透視畸變，適合科學視覺化）
// 4. 縮放 zoom
```

### 三種渲染風格

1. **Surface（表面）**：填充每個 quad，加同色描邊（彩色熱度圖）
2. **Wireframe（線框）**：只繪 quad 邊線（高透明感）
3. **Contour（等高）**：依高度切 10 級階梯著色

### 顏色映射
HSL 熱度圖：低 z → 藍 (240°)，高 z → 紅 (0°)
```
hue = 240° - 240° × zNorm
```

### 互動功能
- **滑鼠拖曳**：旋轉 (rotX, rotY)
- **滾輪**：縮放 (0.2x ~ 5x)
- 32×32 解析度（1024 個面）

### 畫家算法
所有面按平均 z 深度排序，後繪覆蓋前繪 → 正確的遮擋關係

### 驗證結果
- ✅ z = sin(x)·cos(y) → 經典馬鞍曲面，呈現完美鞍點
- ✅ z = sin(√(x²+y²)) → 同心圓波紋，線框模式清晰可辨
- ✅ z 範圍自動計算為 [-1.00, 1.00]
- ✅ 三軸標示（X 紅、Y 綠、Z 藍）正確
- ✅ 旋轉資訊即時更新
