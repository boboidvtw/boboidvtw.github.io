🔒 隱私與安全

您的資料永遠留在您的設備上。
∑ Calc 沒有後端伺服器、沒有資料庫、不收集任何個人資訊。所有計算都在瀏覽器端完成，歷史紀錄和設定僅儲存於您的 localStorage 中。

🛠️ 技術架構

∑ Calc 的核心設計哲學是「單一檔案、零依賴」：

  • 純 HTML + CSS + JavaScript，不使用任何前端框架
  • 自製 3D 引擎：使用 Canvas 2D API 實現旋轉投影與畫家算法
  • 數值方法：中央差分法（微分）、Simpson 1/3 法則（積分）、Bisection 二分法（求根）
  • 響應式設計：桌面與手機端自動適配，CSS Variables 驅動深/淺色主題
  • PWA 支援：可安裝到手機桌面，離線使用
  • 多語系：繁中、英文、簡中、日文完整 i18n（100+ 翻譯鍵）
