三角函數速查表與計算機使用教學 (Sin, Cos, Tan)

三角函數（Trigonometric Functions）是數學、物理、建築與工程學中最基礎也最重要的工具之一。無論是計算直角三角形的邊長、解析物理學中的力學向量，甚至是製作遊戲中的角色移動，都會用到 Sin（正弦）、Cos（餘弦） 和 Tan（正切）。

本文將為你快速複習三角函數的核心概念，並提供常用的角度速查表，最後教你如何使用 ∑ Calc 超強萬能計算機 輕鬆搞定所有複雜的三角函數計算。

1. 什麼是三角函數？(SOH-CAH-TOA)

在一個直角三角形中，如果我們鎖定一個非直角的角度 θ，我們可以根據它與三條邊的關係，定義出三個基本的三角函數。口訣是大家熟悉的 SOH-CAH-TOA：

  • SOH (Sin)：$$\sin(\theta) = \frac{\text{對邊 (Opposite)}}{\text{斜邊 (Hypotenuse)}}$$
  • CAH (Cos)：$$\cos(\theta) = \frac{\text{鄰邊 (Adjacent)}}{\text{斜邊 (Hypotenuse)}}$$
  • TOA (Tan)：$$\tan(\theta) = \frac{\text{對邊 (Opposite)}}{\text{鄰邊 (Adjacent)}}$$

2. 常用特殊角度速查表 (Rad / Deg)

在考試或快速推算時，我們經常會遇到幾個特殊的角度。以下整理了 0°、30°、45°、60° 與 90° 的精確值：

角度 (Degree) 弧度 (Radian) Sin (正弦) Cos (餘弦) Tan (正切)
0° 0 0 1 0
30° π / 6 1/2 $$\frac{\sqrt{3}}{2}$$ $$\frac{\sqrt{3}}{3}$$
45° π / 4 $$\frac{\sqrt{2}}{2}$$ $$\frac{\sqrt{2}}{2}$$ 1
60° π / 3 $$\frac{\sqrt{3}}{2}$$ 1/2 $$\sqrt{3}$$
90° π / 2 1 0 無定義 (∞)

3. 弧度 (Radian) 與角度 (Degree) 的差異

💡 重要觀念：
大多數高階數學與程式語言（包括 JavaScript、Python 的 Math 庫，以及 ∑ Calc 的預設模式）都是使用「弧度 (Radian)」來計算三角函數的。

弧度與角度的換算公式非常簡單：

  • 角度轉弧度：弧度 = 角度 × (π / 180)
  • 弧度轉角度：角度 = 弧度 × (180 / π)

4. 如何在 ∑ Calc 中計算三角函數？

手算 30° 或 45° 很容易，但如果遇到像 sin(37°) 或 cos(1.5 rad) 這種非特殊角怎麼辦？這時候就輪到計算機出場了！

在 ∑ Calc 超強萬能計算機 中，你可以直接在「計算」面板輸入表達式：

範例 A：計算弧度值

若你想計算 sin(π/4)，直接在 ∑ Calc 輸入：

sin(pi/4)

系統會瞬間計算出結果：0.707106781187（即 $$\frac{\sqrt{2}}{2}$$ 的小數近似值）。

範例 B：計算角度值 (Degree)

因為 ∑ Calc 預設使用弧度，如果你想計算 30度 的 sin 值，你需要手動乘上轉換系數 pi/180：

sin(30 * pi/180)

計算結果為 0.5，非常精確！

範例 C：反三角函數 (ArcSin / ArcCos)

如果你知道兩邊的長度，想反推角度，可以使用反三角函數。例如已知 sin(θ) = 0.5，求 θ：

asin(0.5)

得出 0.523598775598 弧度（即 π/6，等於 30°）。
