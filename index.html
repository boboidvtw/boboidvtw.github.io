📖 使用說明
🧮 計算機操作 ▼
  • 點擊數字與運算符號進行計算，按 = 取得結果
  • 分類按鈕：數學 / 工程 / 科學 / 物理 / 進位，切換不同按鈕組
  • 支援函數：sin cos log sqrt ln exp 等
  • 次方用 ^，例如 2^10 = 1024
  • 階乘用 n!，組合數用 nCr(n,r)，排列數用 nPr(n,r)
  • 科學記號用 EE，例如 1.5EE3 = 1500
  • 進位換算：切到「進位」分類，可在 DEC / BIN / OCT / HEX 間互換
  • 按 儲存 可將結果存到「儲存」tab
📐 單位換算 ▼
  • 選擇分類（長度 / 重量 / 溫度 / 面積 / 體積 / 速度 / 壓力 / 能量）
  • 在上方輸入數值並選擇來源單位，下方自動顯示換算結果
  • 下方也可輸入，會反向換算到上方（雙向）
  • 按 ⇅ 可交換上下兩個單位
  • 下方清單顯示該分類所有單位的換算值，點擊可設為目標單位
  • 溫度支援 °C / °F / K 三者互換
  • 面積包含台灣常用的 坪；重量包含 台斤、台兩、台錢
💱 即時匯率 ▼
  • 以 新台幣（TWD） 為基準，顯示 16 種主要貨幣的即時匯率
  • 在新台幣欄輸入金額 → 下方自動換算所有幣別
  • 當前貨幣：選擇外幣並輸入金額 → 台幣欄自動換算（雙向）
  • 點擊下方幣別卡片，可快速切換「當前貨幣」選單
  • 按 ⇒ 可將台幣金額帶入計算機繼續運算
  • 匯率資料來自 ExchangeRate-API，每小時更新一次
  • 按 🔄 可手動刷新匯率
📚 公式庫 ▼
  • 內建 100 條公式，涵蓋 數學 / 物理 / 金融 / 工程 / 健康 / 科學 六大分類
  • 點擊分類 tab 篩選，或在搜尋欄輸入名稱/變數/分類關鍵字
  • 點擊公式卡片 → 跳出輸入框，每個變數都有中文說明
  • 計算後結果自動帶入計算機並存入「儲存」
  • 金融公式的利率欄位輸入年利率 %（如 3），程式自動換算
✏️ 自建公式 ▼
  • 切到「✏️ 自建」tab，填入公式名稱、表達式、選擇分組
  • 表達式使用變數字母（如 P r n），計算時會逐一詢問數值
  • 支援所有內建函數：sin sqrt log ^ 等
  • 自建公式會出現在對應分組 tab 的下方「✏️ 自建公式」區塊
  • 公式存於瀏覽器本地（localStorage），關閉頁面不會消失
  • 點 ✕ 可刪除單條自建公式
💡 其他功能 ▼
  • 儲存 tab：保存最近 10 筆計算結果，點擊可帶回計算機
  • 深色 / 淺色主題：點右上角 ☀️ / 🌙 切換
  • 語言切換：支援繁體中文 / 英文 / 簡體中文 / 日文
  • 所有設定（主題、語言、儲存紀錄、自建公式、匯率快取）均存於本地，不需登入

📖 ∑ Calc 終極使用指南與計算原理

1. 基礎與科學計算

除了加減乘除，∑ Calc 提供了完整的科學與工程計算功能。支援 sin、cos、tan、自然對數 ln 與常用對數 log。使用 ^ 進行次方運算（例如 2^10 = 1024）。在處理極大或極小的物理數值時，您可以使用 EE 代表科學記號（例如 1.5EE3 = 1500）。內建階乘 n!，以及機率統計常用的組合數 nCr 與排列數 nPr。

2. 進位制換算

在 ∑ Calc 中切換到「進位」分類，您可以自由在 DEC (十進位)、BIN (二進位)、OCT (八進位) 與 HEX (十六進位) 之間無縫換算，適合軟體工程師與資訊科學學生使用。

3. 單位換算與即時匯率

內建長度、重量、溫度、面積等八大類別。特別支援台灣常見的「坪」（1 坪 ≈ 3.305785 平方公尺）與「台斤」（1 台斤 = 0.6 公斤）。透過串接 ExchangeRate-API，我們提供每小時更新一次的即時匯率資料，您可以一鍵將換算出的金額帶回計算機中繼續運算。

4. 內建公式與自建公式

涵蓋數學、工程、科學、物理、金融、健康六大領域共 100 條公式。如果您需要特定的計算（例如房貸本息攤還），可以直接使用。您更可以在「✏️ 自建」分頁輸入自己的專屬公式，資料會安全地保存在您的裝置本地（Local Storage），無須擔心隱私外洩。v3.5.5 新增 20 條熱門公式，v3.5.6 再增 5 條健康公式（TDEE、體脂率、水分需求），v3.5.9 再擴充 12 條使公式庫正式突破 100 條，覆蓋國高中數學競賽常考、健康族群常用、創業財管必備。
